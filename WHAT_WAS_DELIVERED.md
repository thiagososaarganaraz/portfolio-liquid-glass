## 🔄 HOW IT WORKS (30-SECOND VERSION)

```
1. User clicks "EN · ES" toggle
   ↓
2. setLang("es") updates React state
   ↓
3. localStorage saves preference
   ↓
4. html lang attribute updates to "es"
   ↓
5. All components re-render with Spanish content
   ↓
6. Result: Spanish text displayed instantly (no reload)
   ↓
7. On next visit: Page loads in Spanish (from localStorage)
```
